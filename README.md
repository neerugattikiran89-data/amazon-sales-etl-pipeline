# Amazon ETL Pipeline

## Overview

This project is a simple ETL (Extract, Transform, Load) pipeline built using Python and Pandas.

The pipeline extracts Amazon dataset data, performs data transformation and cleaning, and generates an output file after processing and loads to MySQl.

---

## Technologies Used

* Python
* Pandas
* MySQL

---

## Features

* Data extraction from dataset
* Data cleaning and transformation
* Processed output file generation
* Basic ETL workflow implementation

---

## Project Workflow

1. Extract data from dataset
2. Transform and clean data
3. Generate processed output file
4. Successfully loaded into Database
